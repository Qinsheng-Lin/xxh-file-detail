# 🚀 重要：如何看到文件信息浏览器

插件已安装成功！但你需要**重新加载窗口**才能看到新功能。

## 第一步：重新加载窗口

请按照以下步骤操作：

1. 按 `Cmd+Shift+P` 打开命令面板
2. 输入：`reload window`
3. 选择：`Developer: Reload Window`
4. 回车执行

## 第二步：查看文件信息浏览器

重新加载后，你会在左侧资源管理器中看到一个新的部分：

```
📁 资源管理器
   └─ XXHFD (你的项目)
   
🔍 搜索

📋 文件信息浏览器  ← ← ← 这是新增的！
   └─ 显示所有文件，带时间和大小
```

## 效果示例

在"文件信息浏览器"中，每个文件后面会显示：

```
📄 api_crawler.py                2025/11/3 20:59, 6.33 kB
📄 DATA_SCRAPING_SUMMARY.md      2025/11/3 20:59, 3.7 kB
📄 HOW_TO_GET_API_QUERY.md       2025/11/3 20:59, 2.46 kB
📁 .vscode
```

## 如果还是看不到

1. 检查插件是否启用：
   - 点击左侧"扩展"图标
   - 搜索 "File Info Decorator"
   - 确保它显示为"已启用"

2. 尝试完全重启VSCode

3. 查看开发者工具（帮助 → 切换开发人员工具），看Console中是否有错误
