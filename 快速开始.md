# 快速开始指南

## 🚀 最快的测试方式（推荐）

1. **在当前VSCode中打开此项目**（如果还没打开）
   ```bash
   code /Users/felix/Desktop/xxhFD
   ```

2. **按 F5 键**
   - 会自动打开一个新的VSCode窗口（扩展开发主机）
   - 在新窗口中，插件已经自动加载

3. **在新窗口中打开任意项目文件夹**
   - 使用 `文件 -> 打开文件夹...` 
   - 或者拖拽一个文件夹到窗口中

4. **查看效果**
   - 看左侧文件树，每个文件后面会显示：`2025/11/3 20:59, 6.33 kB`
   - 鼠标悬停在文件上可以看到更详细的信息

## 📋 功能说明

插件会自动在文件树中显示：
- **修改时间**：如 `2025/11/3 20:59`
- **文件大小**：如 `6.33 kB`

## ⚙️ 自定义设置

在扩展开发窗口中：
1. 打开设置：`Cmd+,` (Mac) 或 `Ctrl+,` (Windows)
2. 搜索 `File Info Decorator`
3. 可以调整：
   - ✅ 是否显示大小
   - ✅ 是否显示日期
   - ✅ 日期格式（默认：`YYYY/M/D HH:mm`）
   - ✅ 是否在文件夹上也显示信息

## 🔧 常用日期格式

| 格式 | 示例输出 |
|------|---------|
| `YYYY/M/D HH:mm` | 2025/11/3 20:59 |
| `YYYY-MM-DD HH:mm:ss` | 2025-11-03 20:59:30 |
| `M/D/YY` | 11/3/25 |
| `MM-DD HH:mm` | 11-03 20:59 |

## 📦 正式安装到VSCode

如果测试满意，想正式安装：

```bash
# 1. 安装打包工具（如果还没安装）
npm install -g @vscode/vsce

# 2. 打包
cd /Users/felix/Desktop/xxhFD
vsce package

# 3. 安装生成的 .vsix 文件
# 在VSCode中：Cmd+Shift+P -> "Extensions: Install from VSIX..."
# 选择生成的 file-info-decorator-1.0.0.vsix
```

## 💡 提示

- 首次启动可能需要几秒钟加载
- 如果没看到效果，尝试重新加载窗口：`Cmd+Shift+P` -> "Developer: Reload Window"
- 修改代码后，在调试窗口点击重启按钮（或 `Cmd+Shift+F5`）即可重新加载

## ❓ 问题排查

### 没有显示文件信息？
1. 确认在"扩展开发主机"窗口中（窗口标题会显示）
2. 检查控制台：`帮助` -> `切换开发人员工具` -> `Console`
3. 应该能看到 "File Info Decorator 插件已激活" 的消息

### 只想显示文件大小不显示日期？
在设置中：
```json
{
  "fileInfoDecorator.showDate": false,
  "fileInfoDecorator.showSize": true
}
```

### 想要文件夹也显示信息？
```json
{
  "fileInfoDecorator.showOnFolders": true
}
```
