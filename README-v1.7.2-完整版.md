### 📋 复制路径（3种方式）

#### 方式1：inline按钮（推荐）
```
1. 鼠标悬停到文件上
2. 点击右边的📋图标
3. 路径已复制
4. 在Chat按Cmd+V粘贴
```

#### 方式2：右键菜单
```
1. 右键文件
2. 选择"复制路径到Chat"
3. 在Chat按Cmd+V粘贴
```

#### 方式3：快捷键
```
1. 选中文件
2. 按Command+Shift+C
3. 在Chat按Cmd+V粘贴
```

## 🎨 选中样式修复

如果选中效果不够明显，参考：`修复选中样式.md`

快速修复：
1. 打开设置：`Command+,`
2. 点击右上角 `{}` 图标
3. 添加：
```json
{
  "workbench.colorCustomizations": {
    "list.activeSelectionBackground": "#0078d4",
    "list.activeSelectionForeground": "#ffffff"
  }
}
```
