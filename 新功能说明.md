# 🎉 新功能 v1.2.0

## ✨ 新增功能

### 1. 📁 显示顶级文件夹
现在会显示工作区的顶级文件夹（如 xxhfd），不是直接展开里面的内容

### 2. 🔍 搜索功能
- 点击标题栏的 **🔍 搜索** 按钮
- 输入文件名关键词（如：`.ts`、`api`、`README`）
- 只显示匹配的文件和文件夹
- 点击 **清除搜索** 按钮恢复显示全部

### 3. 📊 排序功能
在标题栏有三个排序按钮：

- **📋 按名称排序** - 文件名 A-Z 排序（默认）
- **⏰ 按时间排序** - 最新修改的文件在前
- **💾 按大小排序** - 最大的文件在前

### 4. 🔄 刷新按钮
手动刷新文件列表

## 📌 使用说明

**重要：需要重新加载窗口！**

1. 按 `Cmd+Shift+P`
2. 输入 `reload window`
3. 选择 `Developer: Reload Window`

## 🎯 标题栏按钮

从左到右：
```
🔍 搜索 | ✖️ 清除 | 📋 名称 | ⏰ 时间 | 💾 大小 | 🔄 刷新
```

## 💡 使用场景

- **找最近修改的文件**：点击"按时间排序"
- **找最大的文件**：点击"按大小排序"
- **搜索特定文件**：点击"搜索"，输入关键词
- **恢复默认显示**：点击"按名称排序"和"清除搜索"

## 📝 示例

**搜索所有 TypeScript 文件：**
1. 点击 🔍 搜索
2. 输入 `.ts`
3. 只显示 .ts 文件

**查找昨天修改的文件：**
1. 点击 ⏰ 按时间排序
2. 查看顶部的文件

**找出最大的文件：**
1. 点击 💾 按大小排序
2. 顶部就是最大的文件
