# 🎉 完整功能说明 v1.3.0

## ✨ 全部功能

### 📁 文件浏览
- ✅ 显示文件修改时间和大小
- ✅ 显示顶级工作区文件夹
- ✅ 展开/折叠文件夹

### 🔍 搜索和排序
- ✅ **搜索文件** - 按名称筛选文件
- ✅ **按名称排序** - A-Z 排序
- ✅ **按时间排序** - 最新文件在前
- ✅ **按大小排序** - 最大文件在前

### 📝 文件操作

#### 标题栏按钮
在"文件信息浏览器"标题栏有以下按钮：
```
🔍 搜索 | ✖️ 清除 | 📋 名称 | ⏰ 时间 | 💾 大小 | 🔄 刷新 | 📄 新建文件 | 📁 新建文件夹
```

#### 右键菜单功能
在任何文件或文件夹上**右键点击**，可以：

1. **新建文件** (仅在文件夹上)
   - 在选中的文件夹下创建新文件
   - 会提示输入文件名
   - 创建后自动打开文件

2. **新建文件夹** (仅在文件夹上)
   - 在选中的文件夹下创建新文件夹
   - 会提示输入文件夹名

3. **重命名**
   - 重命名文件或文件夹
   - 会显示当前名称，可直接修改

4. **删除**
   - 删除文件或文件夹
   - 会弹出确认对话框，防止误删

5. **复制路径**
   - 复制文件/文件夹的完整路径到剪贴板

6. **在文件管理器中显示**
   - 在系统文件管理器（Finder/资源管理器）中打开

### 🎯 拖放功能

**拖动文件/文件夹移动位置：**
1. 点击并按住文件/文件夹
2. 拖动到目标文件夹
3. 松开鼠标完成移动
4. 如果目标已存在同名文件，会提示是否覆盖

**支持：**
- ✅ 文件移动到文件夹
- ✅ 文件夹移动到另一个文件夹
- ✅ 多选拖动（按住 Cmd/Ctrl 选择多个）
- ✅ 自动检测冲突并提示

## 🎮 使用方法

### 快速创建文件
**方法1：** 点击标题栏的 📄 按钮
**方法2：** 在文件夹上右键 → "新建文件"

### 快速删除文件
**方法1：** 在文件上右键 → "删除"
**方法2：** 选中文件，按 `Delete` 键（即将支持）

### 批量移动文件
1. 按住 `Cmd` (Mac) 或 `Ctrl` (Windows) 选择多个文件
2. 拖动到目标文件夹
3. 松开完成移动

### 查找大文件
1. 点击 💾 "按大小排序"
2. 最大的文件会排在最前面

### 查找最新修改的文件
1. 点击 ⏰ "按时间排序"
2. 最新修改的文件会排在最前面

## ⚠️ 重要提示

**必须重新加载窗口才能看到新功能！**

1. 按 `Cmd+Shift+P`
2. 输入 `reload window`
3. 选择 `Developer: Reload Window`

## 🔥 实用场景

### 场景1：整理项目文件
1. 按时间排序，找到最近修改的文件
2. 拖动到对应的文件夹分类

### 场景2：快速创建组件
1. 在 components 文件夹上右键
2. 选择"新建文件"
3. 输入 `Button.tsx`
4. 自动打开文件开始编码

### 场景3：清理大文件
1. 按大小排序
2. 找到占用空间大的文件
3. 右键删除不需要的

### 场景4：复制文件路径
1. 在文件上右键
2. 选择"复制路径"
3. 粘贴到终端或其他地方使用

## 🎨 自定义设置

按 `Cmd+,` 打开设置，搜索 `File Info Decorator`：

```json
{
  "fileInfoDecorator.showSize": true,
  "fileInfoDecorator.showDate": true,
  "fileInfoDecorator.dateFormat": "YYYY/M/D HH:mm",
  "fileInfoDecorator.showOnFolders": false
}
```
