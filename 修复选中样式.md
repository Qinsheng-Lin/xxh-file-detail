# 🎨 修复选中样式 - 让蓝色更明显

## 📋 问题

文件选中后，背景色不够明显，看不出是蓝色高亮。

## ✅ 解决方法

### 方法1：修改Cursor设置（推荐）

1. 打开Cursor设置：`Command+,`
2. 点击右上角的 `{}` 图标（打开settings.json）
3. 添加以下配置：

```json
{
  "workbench.colorCustomizations": {
    "list.activeSelectionBackground": "#0078d4",
    "list.activeSelectionForeground": "#ffffff",
    "list.inactiveSelectionBackground": "#37373d",
    "list.inactiveFocusBackground": "#37373d",
    "list.hoverBackground": "#2a2d2e",
    "list.focusBackground": "#0078d4",
    "list.focusForeground": "#ffffff"
  }
}
```

4. 保存文件
5. 重新加载窗口：`Command+Shift+P` → `Reload Window`

运行以下命令自动添加配置：

1. `Command+Shift+P`

## 📝 完整配置示例

这是一个完整的settings.json示例，包含了所有相关颜色：

```json
{
  "workbench.colorTheme": "Dark+ (default dark)",
  "workbench.colorCustomizations": {
    // 列表选中颜色
    "list.activeSelectionBackground": "#0078d4",
    "list.activeSelectionForeground": "#ffffff",
    "list.inactiveSelectionBackground": "#37373d",
    "list.inactiveFocusBackground": "#37373d",
    
    // 悬停和焦点
    "list.hoverBackground": "#2a2d2e",
    "list.hoverForeground": "#cccccc",
    "list.focusBackground": "#0078d4",
    "list.focusForeground": "#ffffff",
    
    // 高亮
    "list.highlightForeground": "#0078d4",
    "list.focusHighlightForeground": "#0078d4"
  }
}
```

## ✅ 快速修复脚本

运行这个脚本自动修复：

```bash
#!/bin/bash

# 快速修复选中样式脚本

echo "🎨 开始修复选中样式..."

SETTINGS_FILE="$HOME/.cursor/User/settings.json"

# 检查文件是否存在
if [ ! -f "$SETTINGS_FILE" ]; then
    echo "创建新的settings.json..."
    mkdir -p "$HOME/.cursor/User"
    echo "{}" > "$SETTINGS_FILE"
fi

# 备份
cp "$SETTINGS_FILE" "$SETTINGS_FILE.backup"
echo "✅ 已备份到: $SETTINGS_FILE.backup"

# 使用jq添加配置（如果有jq）
if command -v jq &> /dev/null; then
    jq '. + {"workbench.colorCustomizations": {
        "list.activeSelectionBackground": "#0078d4",
        "list.activeSelectionForeground": "#ffffff",
        "list.inactiveSelectionBackground": "#37373d",
        "list.hoverBackground": "#2a2d2e"
    }}' "$SETTINGS_FILE" > "$SETTINGS_FILE.tmp"
    mv "$SETTINGS_FILE.tmp" "$SETTINGS_FILE"
    echo "✅ 配置已添加！"
else
    echo "⚠️  请手动添加配置"
    echo "打开: $SETTINGS_FILE"
    echo '添加: "workbench.colorCustomizations": {...}'
fi

echo ""
echo "🎯 下一步："
echo "  1. 重新加载Cursor: Command+Shift+P → Reload Window"
echo "  2. 测试选中效果"
echo ""
```

保存为 `fix-selection-color.sh` 并运行：
```bash
chmod +x fix-selection-color.sh
./fix-selection-color.sh
```
