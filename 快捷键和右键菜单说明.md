# ⌨️ 快捷键和右键菜单说明 v1.4.0

## 🎉 新增功能

现在支持完整的右键菜单和键盘快捷键，和VSCode原生资源管理器一样！

## ⌨️ 键盘快捷键

在"文件信息浏览器"中聚焦时，可以使用以下快捷键：

| 快捷键 (Mac) | 快捷键 (Windows/Linux) | 功能 |
|-------------|----------------------|------|
| `Cmd+Backspace` | `Delete` | 删除文件/文件夹 |
| `Enter` | `Enter` | 重命名 |
| `Cmd+C` | `Ctrl+C` | 复制 |
| `Cmd+X` | `Ctrl+X` | 剪切 |
| `Cmd+V` | `Ctrl+V` | 粘贴 |
| `Cmd+Shift+C` | `Ctrl+Shift+C` | 复制路径 |
| `Cmd+Option+C` | `Ctrl+Alt+C` | 复制相对路径 |
| `Cmd+Option+R` | `Ctrl+Alt+R` | 在Finder中显示 |

## 🖱️ 右键菜单功能

### 文件右键菜单
在文件上右键，会显示以下选项：

#### 导航
- **在侧边打开** - 在编辑器右侧打开文件
- **在文件管理器中显示** - 在Finder/资源管理器中定位

#### 比较
- **选择以进行比较** - 选择此文件作为比较源
- **与已选项目比较** - 和之前选择的文件进行对比

#### 编辑
- **剪切** (`Cmd+X`) - 剪切文件，可粘贴到其他位置
- **复制** (`Cmd+C`) - 复制文件

#### 路径
- **复制路径** (`Cmd+Shift+C`) - 复制完整路径
- **复制相对路径** (`Cmd+Option+C`) - 复制相对于工作区的路径

#### 修改
- **重命名** (`Enter`) - 重命名文件
- **删除** (`Cmd+Backspace`) - 删除文件

#### 搜索
- **查找文件中的所有引用** - 查找代码引用

### 文件夹右键菜单
在文件夹上右键，会显示以下选项：

#### 导航
- **在文件管理器中显示** - 在Finder/资源管理器中打开
- **在集成终端中打开** - 在此文件夹打开终端

#### 编辑
- **剪切** (`Cmd+X`) - 剪切文件夹
- **复制** (`Cmd+C`) - 复制文件夹
- **粘贴** (`Cmd+V`) - 粘贴文件到此文件夹

#### 路径
- **复制路径** - 复制完整路径
- **复制相对路径** - 复制相对路径

#### 创建
- **新建文件** - 在此文件夹下创建新文件
- **新建文件夹** - 在此文件夹下创建新文件夹

#### 修改
- **重命名** (`Enter`) - 重命名文件夹
- **删除** (`Cmd+Backspace`) - 删除文件夹及其内容

## 🎯 使用场景

### 场景1：快速删除文件
1. 选中文件
2. 按 `Cmd+Backspace` (Mac) 或 `Delete` (Windows)
3. 确认删除

### 场景2：复制文件到另一个文件夹
1. 在源文件上按 `Cmd+C`
2. 在目标文件夹上右键
3. 选择"粘贴"或按 `Cmd+V`

### 场景3：移动文件（剪切）
1. 在文件上按 `Cmd+X`
2. 在目标文件夹上按 `Cmd+V`

### 场景4：比较两个文件
1. 在第一个文件上右键 → "选择以进行比较"
2. 在第二个文件上右键 → "与已选项目比较"
3. 会打开对比视图

### 场景5：在侧边同时查看两个文件
1. 在第一个文件上单击（正常打开）
2. 在第二个文件上右键 → "在侧边打开"
3. 两个文件并排显示

### 场景6：在终端中操作文件
1. 在文件夹上右键
2. 选择"在集成终端中打开"
3. 终端会切换到该文件夹

### 场景7：快速重命名
1. 选中文件/文件夹
2. 按 `Enter` 键
3. 输入新名称

### 场景8：复制路径用于终端命令
1. 在文件上右键
2. 选择"复制路径"
3. 粘贴到终端或其他地方使用

## 💡 提示

1. **批量操作**：
   - 按住 `Cmd` (Mac) 或 `Ctrl` (Windows) 可以选择多个文件
   - 然后可以一起剪切、复制或删除

2. **快捷键组合**：
   - `Cmd+C` 然后 `Cmd+V` = 复制文件
   - `Cmd+X` 然后 `Cmd+V` = 移动文件
   - 选中 + `Enter` = 重命名

3. **相对路径 vs 绝对路径**：
   - 绝对路径：`/Users/felix/Desktop/xxhFD/src/extension.ts`
   - 相对路径：`src/extension.ts`（相对于工作区）

## 🔄 重要：重新加载窗口

安装后必须重新加载窗口才能使用新功能！

1. 按 `Cmd+Shift+P`
2. 输入 `reload window`
3. 回车
