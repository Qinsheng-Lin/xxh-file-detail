# 调试步骤

## 1. 检查插件是否激活

1. 按 `Cmd+Shift+P`
2. 输入：`Developer: Show Running Extensions`
3. 查找 "File Info Decorator"，确认它在列表中

## 2. 查看开发者工具

1. 点击菜单：`帮助` → `切换开发人员工具`
2. 在 Console 标签中查看是否有：
   ```
   File Info Decorator 插件已激活
   ```
3. 如果有错误信息，截图给我

## 3. 确保打开了工作区文件夹

插件需要一个打开的文件夹才能工作：
- 确认你已经用 `文件` → `打开文件夹...` 打开了项目

## 4. 查找"文件信息浏览器"视图

在左侧边栏：
1. 点击第一个图标（资源管理器）
2. 向下滚动
3. 应该能看到 "文件信息浏览器" 部分

## 5. 如果还是看不到

尝试手动展开：
1. 在资源管理器标题栏右侧，点击 `...` （更多操作）
2. 看是否有选项可以显示/隐藏视图

## 6. 完全卸载重装

```bash
code --uninstall-extension your-name.file-info-decorator
code --install-extension /Users/felix/Desktop/xxhFD/file-info-decorator-1.1.0.vsix
```

然后重启VSCode（不是重新加载，是完全退出再打开）
